# Auto-SPA
## This project is an example of a Single Page Application.

## Technologies used:

* Backend:
  * Database: OracleXE 11g
  * Server: NodeJS
* Client:
  * Builder: Gulp
  * JS libraries:
    * Backbone
    * JQuery
    * Underscore templates
  * JS modules: RequireJS
  * Styles: Sass

## Installation

* Configure the database (see `db` folder)
* Clone this repository
* `npm i`
* `bower i`
* `gulp`
* `node app.js`
